## 🚀 部署

### GitHub Pages
1. Fork 此仓库
2. Settings → Pages → Source 选择 `main` 分支
3. 保存，等待几分钟即可访问

### Vercel
1. 导入此仓库到 Vercel
2. Framework Preset: **Other**
3. 直接部署，零配置
